# Operator Docs

`docs/reference/operator/` groups the runbooks and operator-facing surfaces for Codexify.

## Sections

- `ops/`: operational protocols, runbooks, and failure signatures
- `help/`: setup and FAQ material
- `cli/`: command references
- `desktop/`: Tauri and desktop parity notes
- `release/`: release-facing docs and smoke-test notes
- `github/`: GitHub and CI-specific docs
- `ui/`: UI-specific operator notes
- `examples/`: example artifacts and sample data
- `debug/`: focused troubleshooting notes
