# Sequence patterns

Reusable scene syntax:

```text
WorkspaceReveal -> FocusPush -> MessageIsolation -> ThreadDive -> BreathingHold
SidebarDrift -> ContextSwitch -> CardLift -> DocumentUnfold -> InterfacePullback
GallerySweep -> ArtifactPortal -> ParallaxWorkspace -> FocusPush -> ColorBridge
WorkspaceReveal -> SidebarDrift -> ThreadDive -> DocumentUnfold -> GallerySweep -> ContextSwitch -> InterfacePullback
```

The final sequence is the default skeleton for Peekaboo Cinematic V2. The current manifest uses a bounded functioning subset and names each shot explicitly. Unsupported grammar entries remain visible in the registry documentation rather than silently degrading to a different shot.
