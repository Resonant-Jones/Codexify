# Codexify Personalized Presentation

## Purpose

This packet defines a Remotion-based presentation for Codexify that is personalized for your own workspace, accent system, assistant name, and imported chats.

It is intentionally structured for short clips that can be recorded individually and spliced together later.

Target output:

- Title: `Codexify Personalized Presentation`
- Duration: `100s` base cut, with each segment designed as a `20s` clip
- Format: `1920x1080`
- Aspect ratio: `16:9`
- Framerate: `30fps`
- Voice over: `none`

## What You Can Prep First

Before recording, prepare these inputs:

- Wallpaper image for the Appearance tab
- Accent color or color pair
- Assistant name you want shown in the UI
- One imported chat project in the `Imports` area
- A short list of feature beats you want to show

If you want the clips to feel cohesive, keep the wallpaper and accent color consistent across all segments.

## Browser Path

Use the local Codexify UI in the in-app browser first.

Use Chrome if you need your logged-in profile, cookies, extensions, or an existing browser session.

Avoid Safari unless there is a specific reason to match Safari-only behavior.

## Source Truth Boundaries

This is a presentation about your actual configured workspace, not a general feature trailer.

Safe present-tense focus:

- You can personalize appearance with wallpaper and accent colors.
- You can set a persona or assistant voice in the Settings/Imprint surface.
- You can import ChatGPT history through the Data import flow.
- You can open and inspect imported project chats inside Codexify.

Do not claim broader runtime guarantees than the screens you actually show.

## Clip Plan

Each clip is designed to stand alone at `20s`, which makes the final edit easy to rearrange.

1. `0:00-0:20` - `Wallpaper and Accent`
   Show the Appearance tab, pick the wallpaper, and tune the base accent system color.
   Caption: `Make the workspace feel like yours.`

2. `0:20-0:40` - `Assistant Identity`
   Move to the Persona/Imprint surface and set the assistant name and voice text.
   Caption: `Name the assistant. Shape the tone.`

3. `0:40-1:00` - `Import Your Chats`
   Open the Data tab, point at the Imports project, and show the upload or imported project state.
   Caption: `Bring your own history into the workspace.`

4. `1:00-1:20` - `Open a Real Thread`
   Jump into one imported conversation and show the thread in context.
   Caption: `Your chats stay readable as project material.`

5. `1:20-1:40` - `End Frame`
   Return to a composed shell view with the personalized wallpaper and accent system visible.
   Caption: `Codexify, tuned to your workflow.`

## Recording Guidance

- Keep each clip focused on one feature only.
- Use slow cursor movement and fewer clicks.
- Wait for each screen change to settle before moving on.
- Avoid voice-over unless you later decide you want narration.
- Let captions do the work.
