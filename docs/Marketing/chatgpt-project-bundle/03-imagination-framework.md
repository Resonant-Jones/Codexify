# Imagination Framework

## Framework Thesis

Strong marketing does not only describe features. It helps the buyer recognize a more capable version of themselves.

Codexify rule:

> Codexify marketing is identity transition, bounded by proof.

## The Five Methods

1. Flip the status
2. Hijack the myth
3. Open the hidden door
4. Build the ritual
5. Give them a superpower

## Codexify Translation

Before:
- scattered AI chats
- fragmented documents
- context loss
- prompt hoarding
- workflow drift
- opaque automation
- unclear system state
- leased intelligence from disconnected SaaS tools

After:
- durable project memory
- visible knowledge boundaries
- retrievable documents
- traceable lineage
- explicit truth surfaces
- reusable workflow thinking
- local-first operating posture
- stronger cognitive continuity

## Operating Principles for Drafting

- Identity-transition principle: write to who the user becomes, not just what features do.
- Proof-boundary principle: never claim beyond current supported truth.
- Continuity principle: favor language about durable intelligence over one-turn novelty.

## Pre-Draft Checklist for Marketing Assistants

1. Anchor every narrative to `01-current-truth-claim-boundary.md`.
2. Pick one method as the primary frame; use others as support.
3. State transformation in before/after form.
4. Mark risky claims as caution/future before channel drafting.
5. Remove any line that sounds stronger than current proof.
