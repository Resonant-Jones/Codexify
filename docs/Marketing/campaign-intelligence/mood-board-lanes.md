# Mood Board Lanes (Internal)

Use these lanes for mood boards, ad mockups, infographic concepts, and brand-awareness drafts.

Global style baseline for Codexify:
- local-first, sovereign, precise
- cinematic but quiet
- dark and slate-forward palette
- subtle glass layers, restrained teal/cyan, warm iron accents
- no neon sci-fi clutter

## Private Library

When to use:
- Knowledge continuity stories
- Document-to-intelligence workflows
- Research and writing audience segments

Emotional job:
- Create calm confidence that memory can be curated and kept

Visual language:
- Low-light reading rooms, shelf-grid layouts, index cards, subtle metadata overlays
- Slate charcoal backgrounds with teal/cyan keylines and warm iron accent marks

Copy flavor:
- "durable knowledge", "private canon", "retrievable context"

Sample asset ideas:
- Before/after panel: scattered notes versus indexed project library
- Homepage hero still: thread + document continuity stack
- Infographic: capture -> retrieve -> reuse loop

Claim risks:
- Implying perfect recall or zero context loss
- Implying automatic understanding without review

## Operator Console

When to use:
- Runtime truth, health surfaces, and governance education
- Technical audiences evaluating operational trust

Emotional job:
- Convey control, observability, and informed action

Visual language:
- Structured panels, timeline strips, status tags, event traces
- Dense but readable dark UI grids with restrained accent color

Copy flavor:
- "inspectable runtime truth", "proof before promise", "state you can verify"

Sample asset ideas:
- Social carousel: accepted -> queued -> completed lifecycle
- Ad mockup: operational dashboard with claim gate overlays
- Infographic: route -> queue -> worker -> persistence chain

Claim risks:
- Suggesting complete observability on all paths
- Collapsing acceptance into completion guarantees

## Memory Spine

When to use:
- Thread/project continuity positioning
- Founder notes and educational continuity narratives

Emotional job:
- Make persistence feel coherent and human-scaled under complexity

Visual language:
- Vertical timeline motifs, linked nodes, artifact breadcrumb trails
- Quiet motion cues that imply accumulation over time

Copy flavor:
- "keep the thread", "continuity under complexity", "intelligence over time"

Sample asset ideas:
- Motion concept: fragments converging into one durable thread
- Educational diagram: project spine with branch recoverability
- Community post visual: weekly ritual continuity board

Claim risks:
- Implying no drift or guaranteed reconstruction
- Overstating cross-tool synchronization guarantees

## Sovereign Stack

When to use:
- Local-first ownership and user-control narratives
- Privacy-conscious and infrastructure-minded audiences

Emotional job:
- Reinforce agency and ownership without paranoia framing

Visual language:
- Layered stack diagrams, boundary boxes, key/capability iconography
- Subtle glass panes over dark slate architecture scenes

Copy flavor:
- "own your AI context", "operate infrastructure you can govern", "local-first by design"

Sample asset ideas:
- One-page stack explainer with trust boundaries
- Ad series: central dependency versus user-owned boundary
- Brand-awareness stills showing device/node boundary maps

Claim risks:
- Implied absolute privacy or security guarantees
- Implied full federation/cloud parity by default

## Firehose Filter

When to use:
- Signal extraction, claim discipline, and anti-noise messaging
- Audiences drowning in tool sprawl and context overload

Emotional job:
- Show relief through selective, evidence-linked intelligence

Visual language:
- High-volume input streams entering a disciplined filter and outputting concise artifacts
- Controlled contrast, restrained highlights, and hierarchy-forward typography

Copy flavor:
- "from noise to signal", "bounded outputs", "evidence-linked decisions"

Sample asset ideas:
- Infographic: raw feed -> claim gate -> campaign packet
- Social graphic: what changed/what matters card system
- Educational post visual: safe/caution/future/reject classifier board

Claim risks:
- Suggesting universal correctness or automated truth guarantees
- Suggesting autonomous publishing without human review
