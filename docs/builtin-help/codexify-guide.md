# Codexify Guide

Codexify is the local-first workspace for projects, documents, chat, and media.

`General` is the default project scope when no explicit project is chosen.

Gallery, Documents, and Chat are related but distinct surfaces:

- Gallery handles media and generated assets.
- Documents holds persistent text artifacts.
- Chat contains conversation threads and turn history.

Guardian can answer questions using available local docs together with thread and project context.
