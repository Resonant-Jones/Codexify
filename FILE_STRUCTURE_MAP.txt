GUARDIAN CODEBASE - FILE STRUCTURE MAP
======================================

BACKEND - COLLABORATION & REALTIME
==================================

1. WebSocket & Collaboration Manager
   /guardian/realtime/collaboration.py (202 lines)
   ├── CollaborationManager class
   │   ├── __init__(): Initialize active connections and presence maps
   │   ├── connect(doc_id, ws, user_id): Register WebSocket connection
   │   ├── disconnect(doc_id, ws, user_id): Remove WebSocket connection
   │   ├── broadcast(doc_id, message): Send to all connected clients
   │   ├── get_active_sessions(): Return session count
   │   └── get_session_user_count(doc_id): Return user count for session
   ├── @router.websocket("/ws/{document_id}"): WebSocket endpoint
   └── Message types: presence.join, presence.leave, update

2. API Routes - Documents & Autosave
   /guardian/routes/documents.py (264 lines)
   ├── POST /api/documents/autosave
   │   ├── Request: AutosaveRequest(thread_id, content)
   │   ├── Response: AutosaveResponse(ok, document_id, relation)
   │   ├── Logic: Create/update GeneratedDocument, ThreadDocument link
   │   └── Event: document.autosave
   └── GET /api/threads/{thread_id}/documents
       ├── Fetch ThreadDocument links
       ├── Retrieve GeneratedDocument details
       └── Return ordered list of documents

3. API Routes - Share Links
   /guardian/routes/share.py (315 lines)
   ├── POST /api/share
   │   ├── Request: CreateShareRequest(target_type, target_id, expires_in_days)
   │   ├── Response: CreateShareResponse(ok, token, url, expires_at)
   │   ├── Logic: Generate token, create SharedLink, optionally set expiry
   │   └── Event: share.created
   └── GET /api/share/{token}
       ├── Fetch SharedLink by token
       ├── Validate expiry
       ├── Fetch content (thread or document)
       ├── Event: share.accessed
       └── Return read-only content

4. API Routes - Workspace
   /guardian/routes/workspace.py (91 lines)
   ├── GET /api/workspace/{thread_id}
   │   ├── Fetch thread metadata
   │   ├── Collect linked documents via ThreadDocument
   │   ├── Fetch workspace diagnostics
   │   └── Return: {thread, documents, diagnostics}
   └── _collect_thread_documents(thread_id): Helper function

DATABASE - ORM MODELS
=====================

/guardian/db/models.py (514 lines)

Core Models:
├── ChatThread
│   ├── id (PK), user_id, title, summary
│   ├── project_id (FK), parent_id (FK, self-ref)
│   ├── archived_at, created_at, updated_at
│   └── Relationships: messages, parent, children
│
├── ChatMessage
│   ├── id (BigInteger, PK), thread_id (FK, CASCADE)
│   ├── role, content, created_at
│   └── Relationship: thread
│
├── GeneratedDocument (UUID, PK)
│   ├── project_id (FK), thread_id (FK), user_id
│   ├── title, content, format, model
│   ├── created_at, updated_at, deleted_at (soft delete)
│   └── Relationships: project, thread
│
├── UploadedDocument (UUID, PK)
│   ├── project_id (FK), thread_id (FK), user_id
│   ├── filename, filesize, mime_type, src_url
│   ├── parsed_text (for FTS), deleted_at
│   └── Relationships: project, thread
│
├── ThreadDocument
│   ├── id (PK), thread_id (FK, CASCADE)
│   ├── document_id (UUID, FK), relation (autosave/attached/reference)
│   ├── created_at
│   └── Indexes: thread_id, (thread_id, relation), document_id
│
├── SharedLink (UUID, PK)
│   ├── target_type (thread/document), target_id
│   ├── token (unique, 64 chars), expires_at (nullable)
│   ├── created_at
│   └── Indexes: token, (target_type, target_id)
│
├── Project
│   ├── id (PK), name (unique), description, icon
│   └── created_at, updated_at
│
├── EventOutbox (BigInteger, PK)
│   ├── topic, payload (JSONB), status
│   ├── tenant_id, created_at
│   └── Indexes: tenant_id, (status, created_at)
│
├── AuditLog (BigInteger, PK)
│   ├── event, entity, entity_id, user_id, timestamp
│   └── Indexes: timestamp DESC, (entity, entity_id)
│
└── Legacy/Other
    ├── UploadedImage, GeneratedImage
    ├── MemoryEntry
    ├── ConnectorConfig, ConnectorRun, RawDocument
    ├── SyncJob, TTSOutput
    └── Message (deprecated, use ChatMessage)

EVENT SYSTEM
============

/guardian/core/event_bus.py (136 lines)

Durable Event Bus:
├── configure_event_store(store: ChatDB)
├── configure_fallback_emitter(emitter)
├── emit_event(topic, payload, tenant_id)
│   ├── Persists to PostgreSQL events_outbox
│   ├── In-memory fanout to subscribers
│   └── Fallback to fallback emitter if no store
├── subscribe_in_memory() -> asyncio.Queue
├── unsubscribe_in_memory(queue)
├── fetch_events_after(last_id, limit)
├── delete_events_through(last_id, tenant_id)
└── is_persistent_enabled() -> bool

Event Topics:
├── collab.update: On client WebSocket update
├── document.autosave: On document autosave
├── share.created: On share link creation
└── share.accessed: On share link retrieval

/guardian/sync/bus.py (31 lines)

Alternative EventBus:
├── EventBus class (simple in-memory)
│   ├── __init__()
│   ├── async publish(message)
│   ├── async subscribe() -> AsyncIterator
│   └── _subscribers: List[asyncio.Queue]
└── bus = EventBus() (global instance)
